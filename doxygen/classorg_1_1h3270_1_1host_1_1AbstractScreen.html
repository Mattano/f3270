◆ getInputFieldAt()

InputField org.h3270.host.AbstractScreen.getInputFieldAt ( final int  x,
final int  y 
)
inline

Returns a Field object representing the input field at position (x,y). If there is no input field at this position, this method returns null. A field begins with the character after the first control character, and ends with the character before the terminating control character. Thus, for the positions of the control characters themselves, this method always returns null.

x and y start in the upper left hand corner, which is position (0,0).
